RollCall+

The current manual attendance system in colleges is problematic due to its time consuming, error prone, and manipulable nature, which can delay attendance data availability and lead to inaccuracies. This can adversely affect student performances and evaluations, and the system lacks transparency and accountability. Online attendance systems were developed to overcome these challenges but they also face issues such as technical glitches, privacy concerns and implementation costs. However, if implemented correctly technology can be a boon for the attendance system and the institution. RollCall+ application aims to provide an IoT based attendance system using Blockchain technology that can provide a secure, reliable and tamper proof attendance tracking system for educational institutions corporate offices and other organizations the system can be used to record and track sentence in real time. It can also provide an immutable record of attendance that can be used for verification and audit purposes. The system can be built using IoT devices such as sensors, cameras, and RFID tags. These devices can be used to collect attendance data from the users, the data can then be stored in a secure blockchain database. This database can be accessed by the authorised personnel in order to verify and audit the attendance data. The system can also be used to provide rewards to employees and students for attendance the rewards can be in the form of points or tokens that can be used to avail discounts or other benefits. this is system can also be used to automate the process of attendance tracking thus eliminating the need for manual intervention. The aim of RollCall+ is to create an attendance tracking system by utilising IoT devices in order to automatically capture attendance data, eliminating the need for manual tracking. This can be implemented in the organizations in the following way: IoT sensors are placed in classrooms or other areas where attendance needs to be tracked. When students enter or exit the room the IoT sensors automatically capture attendance data and send it to the attendance tracking software. the attendance tracking software integrates the IoT data with blockchain technology ensuring the that the data is secure, immutable and transparent. The attendance data is stored on a decentralised blockchain Ledger, which can be accessed by authorised parties including faculty and administrators. Any changes to the attendance data are recorded on the blockchain creating a permanent record of all attendance transactions. The attendance tracking system can provide real time attendance data which can be used to improve attendance management commerce student engagement and academic performance. The attendance tracking system can also generate reports common notifications and at the words based on attendance data, providing valuable insights to faculty and administrators. Overall, an IoT based attendance system using blockchain technology provides a secure efficient and effective way to track attendance reduce fraud and improve the educational experiences for students and faculty alike.
